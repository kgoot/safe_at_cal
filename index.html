WalkMe

Let WalkMe find you the best route home. Check out your options all in one place: Uber, lyft, Cal safety shuttles, or walk with another student going along your route. Our route recommendation is based on recent crime data, current traffic patterns, and the locations of Berkeley street lights.

  • Tell Me More

Walk

Unlike other mapping apps, WalkMe won't necessarily have you take the fastest route. Instead, it will map you away from traditionally dangerous, dimly lit areas.

Ride

Know your options. How do lyft and Uber prices compare? How long until the next safety shuttle? Find all the info you need in one place.

Meet up

Tell WalkMe where you are going and it will connect you with other Berkeley students walking a similar route.

Set Up

Our app is currently in approval phases with Apple. In the meantime, since our code is open sourced, it is possible to set up a beta version of the app by cloning the code from the git repository git repository. This is how we’ve set up our app for all our beta users.

  1. Run git clone https://github.com/kgoot/safe_at_cal.git
  2. Connect your iPhone to your computer.
  3. Open the project repository in XCode by navigating to the Xcode Menu, clicking Open another project... and selecting the WalkMe.XCodeProj directory from the cloned git repository.
  4. Navigate to the general settings page and under signing click the Team dropdown menu.
  5. Select Add an account and sign into your Apple account. Make sure the signing certificate updates.
  6. Open your phone, navigate to settings → general → device management → and approve the new Apple account set up above as a developer app.
  7. Clicking the run button back in Xcode and selecting the iphone device that’s plugged in will install the app on the current phone.
